# simpleAPI

![Project Screenshot](./images/simpleAPI.jpeg "simpleAPI")

### Libraries used
- .net
- Swagger
- FluentValidation
- log4net

### Objectives

Build a api with .net with:
- logging
- verification
- deploy to podman

### Run

- dotnet run
- example: http://localhost:5214/swagger/index.html
